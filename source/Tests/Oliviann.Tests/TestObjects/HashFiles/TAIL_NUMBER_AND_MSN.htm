Tail Number And MSN

 

 

 

The IETM identifies aircraft by the property tail number ( TAILNO). Tailno is an integer seven digit numeric. The tail number AF61-02358 is identified as 6102358 internal to the IETM. The following Table identifies tail numbers, cross reference to manufacturer serial number ( MSN), Model and Coverage which identifies each aircraft in IETM. Aircraft coverage identified with LIMITED are aircraft that have been placed in inactive status  and are only covered in the IETM to that point.
